## Project Overview

This project applies Decision Tree and Ensemble Methods (Random Forest, AdaBoost, and Stacking) to predict student performance using the Student Exam Performance dataset.

The objective is to classify students into:

* Pass (1)
* Fail (0)

## Methods Used

* Decision Tree
* Random Forest (Bagging)
* AdaBoost (Boosting)
* Stacking Classifier

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-score
* Confusion Matrix

## Key Findings

Ensemble methods outperformed the Decision Tree. Random Forest and Stacking provided better generalization and higher prediction performance.
